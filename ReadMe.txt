Documentation:
1)cd
    changes current working directory
    Syntax: cd [directory_name]
            or cd .. (to go to parent directory)
            or cd ../ .. 
2)history
    shows a list of all the commands implemented uptil now
    Syntax: history
3)pwd
    displays current working directory 
    Syntax: pwd
4)echo
    prints the text without any quotes
    Syntax: echo "Enter your text here"
            echo Enter your text here

5)exit
    exits the shell
    Syntax: exit

6)All external commmands from bin 
    Can only take 3 arguments with command

CORNER CASES:
    ignores extra spaces in commands
    gives error message if command isn't present in bin
    to access a new shell in shell, the bin file has to be moved to present working directory
